- 절대 길이
	- px - pixels (1px = 1/96th of in)
	- pt - points (1pt = 1/72 of in) windows = 12px mac = 9px
	- ...

- 상대 길이
	- % - 부모 값에 대해서 백분률로 환산한 크기
	- em - font-size를 기준으로 값을 환산. 소수점 3자리 까지 표현가능
	- rem - root의 font-size를 기준으로 값을 환산
	- vw - viewport의 width값을 기준으로 1%의 값으로계산
	- ...
